# My-Starship-Prompt

#### Get started
```
brew install starship
```
```
sudo echo "eval '$(starship init zsh)'" >> ~/.zshrc
```
```
source .zshrc
```
```
# Paste my config into this file then

sudo nano ~/.config/starship.toml
```
